---注册小程序（大陆/海外）---
 
* 账号状态：已激活，未填入个人信息，不能使用

1. 海外（只能注册企业: 个体工商户/企业）
    申请需要企业号码和当地手机号验证
2. 中国大陆（可申请个人、企业、政府、媒体、其他）
    申请个人需要身份证号和国内手机号验证
